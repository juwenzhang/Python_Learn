## 集合的操作

> python 中使用的 **{}** 来定义我们的集合，是一种可变的数据类型
>
> 集合的是一个无序并且内部元素不重复的数据容器
>
> * 无序是只我们的每个元素是没有任何的下标来进行指代的
> * 集合只能是一种不可变的数据类型才可以作为集合的元素构成



> 集合的操作：
>
> * **.add(element)** 向集合添加元素
> * **.clear()** 实现的就是清空集合中的元素
> * **.remove()** 实现的就是移除集合中的元素
> * **set()** 就是实现的是将其他的数据类型强制性的转换为集合类型的数据
> * **in** 实现的是判断某种数据值是否存在于集合中，返回 bool 值
> * **|** 实现的就是求解两个集合的并集
> * **&** 实现的就是求取两个集合的交集
